# react-native-turbo-battery

React Native TurboModule for getting battery level.

## Installation

```sh
npm install react-native-turbo-battery
```

## Usage

```js
import { multiply } from "react-native-turbo-battery";

// ...

const result = await multiply(3, 7);
```
