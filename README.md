# react-native-turbo-battery

React Native TurboModule for getting battery level.

## Installation

```sh
yarn add react-native-turbo-battery
```

## Usage

```js
import { getBatteryLevel } from "react-native-turbo-battery";

// ...

const batteryLevel = getBatteryLevel();
```
